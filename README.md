# MyChaty

Messaging web application (dev)

## Features 
- **User Authentication**
  - Secure user login and registration (Email and Password)

- **Real-Time Messaging**
  - Instant messaging with real-time updates

- **Group Chats**
  - Create and manage group chats
  - Add or remove members from group chats
